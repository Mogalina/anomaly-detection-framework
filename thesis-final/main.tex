% Abstract
\begin{abstract}
\noindent
Modern distributed systems, composed of microservices and heterogeneous
components, are increasingly complex and prone to performance anomalies that can
compromise reliability and service quality. This thesis presents a comprehensive
anomaly detection framework designed for large-scale distributed systems,
integrating real-time edge detection via an LSTM-Autoencoder, federated learning
for collaborative model training without centralizing raw telemetry,
causal graph-based root cause analysis, and adaptive Q-learning thresholding.
By combining localized anomaly detection at the edge with global model
aggregation via the FedAvg algorithm, the framework enhances detection accuracy
while preserving data privacy through Differential Privacy (DP-SGD).
Experimental evaluation on the Server Machine Dataset (SMD), Train-Ticket, and
Alibaba-derived topologies demonstrates that the federated LSTM-AE achieves
competitive detection fidelity, the DP mechanism exposes a sharp
privacy--utility phase transition governed by clipping norm, and causal RCA
provides operationally useful root-cause rankings on small topologies. The
Q-learning threshold controller reduces final alert volume but exhibits
cold-start instability, motivating future reward-shaping improvements. The
thesis reports both positive results and identified limitations to guide future
work.
\end{abstract}
